## Updating Screenshots

Drop new screenshots into `MailTub/` then copy to this folder:

```bash
# From repo root
cp MailTub/banner.png          mailtub/docs/screenshots/banner.png
cp MailTub/banner-dark.png     mailtub/docs/screenshots/banner-dark.png
cp MailTub/full_ui_dark.png    mailtub/docs/screenshots/full-ui-dark.png
cp MailTub/inbox-dark.png      mailtub/docs/screenshots/inbox-dark.png
cp MailTub/inbox-light.png     mailtub/docs/screenshots/inbox-light.png
cp MailTub/email-detailed.png  mailtub/docs/screenshots/email-detail.png
cp MailTub/settings.png        mailtub/docs/screenshots/settings.png
cp MailTub/mobile-pwa.png      mailtub/docs/screenshots/mobile-pwa.png
cp MailTub/admin-dark.png      mailtub/docs/screenshots/admin-dark.png
cp MailTub/admin-light.png     mailtub/docs/screenshots/admin-light.png
```
